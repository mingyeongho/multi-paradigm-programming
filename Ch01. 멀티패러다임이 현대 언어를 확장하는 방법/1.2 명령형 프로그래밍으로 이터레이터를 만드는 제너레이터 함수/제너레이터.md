## 제너레이터

반복자 패턴인 **이터레이터**를 **명령형** 코드로 구현하고 생성할 수 있는 도구

`function*` 키워드로 정의

`next()`를 호출하면 `yield` 키워드를 만날 때까지 실행.

`yield`를 통해 외부로 값을 반환하고 이후 `next()`를 다시 호출하면 이전 실행 지점부터 이어서 함수가 재개

```ts
function* generator() {
  let number = 1;
  const age = 28;
  yield number;

  number++;
  yield { number, age };

  number *= 4;
  yield number;

  number **= 2;
  yield number;

  number = 0;
  yield number;
}

const iter = generator();

console.log(iter.next()); // { done: false, value: 1 }
console.log(iter.next()); // { done: false, value: { number: 2, age: 28 } }
console.log(iter.next()); // { done: false, value: 8 }
console.log(iter.next()); // { done: false, value: 64 }
console.log(iter.next()); // { done: false, value: 0 }
console.log(iter.next()); // { done: false, value: undefined }
```

✅ 함수 **내부 상태를 유지**하면서 순차적으로 값을 반환할 수 있다.

```ts
// lib.es2015.generator.d.ts
interface Generator<
  T = unknown,
  TReturn = any,
  TNext = any,
> extends IteratorObject<T, TReturn, TNext> {
  // NOTE: 'next' is defined using a tuple to ensure we report the correct assignability errors in all places.
  next(...[value]: [] | [TNext]): IteratorResult<T, TReturn>;
  return(value: TReturn): IteratorResult<T, TReturn>;
  throw(e: any): IteratorResult<T, TReturn>;
  [Symbol.iterator](): Generator<T, TReturn, TNext>;
}

// lib.es2015.iterable.d.ts
/**
 * Describes an {@link Iterator} produced by the runtime that inherits from the intrinsic `Iterator.prototype`.
 */
interface IteratorObject<
  T,
  TReturn = unknown,
  TNext = unknown,
> extends Iterator<T, TReturn, TNext> {
  [Symbol.iterator](): IteratorObject<T, TReturn, TNext>;
}
```

### yield\* 키워드

제너레이터 함수 안에서 이터러블을 순회하며 해당 이러러블이 제공하는 요소들을 순차적으로 반환하도록 해준다.

```ts
function* gen() {
  yield 1;

  yield* [2, 3];

  yield 4;
}

const ite = gen();

console.log(ite.next()); // { done: false, value: 1 }
console.log(ite.next()); // { done: false, value: 2 }
console.log(ite.next()); // { done: false, value: 3 }
console.log(ite.next()); // { done: false, value: 4 }
console.log(ite.next()); // { done: true, value: undefined }
```

### naturals 제너레이터 함수

자연수를 무한히 생성하는 함수

```ts
function* naturals() {
  let num = 1;
  while (true) {
    yield num++;
  }
}
```

### 제너레이터로 작성된 reverse 함수

```ts
function* reverse<T>(arrayLike: ArrayLike<T>) {
  let idx = arrayLike.length;
  while (idx) {
    yield arrayLike[--idx];
  }
}

const array = [1, 2, 3, 4, 5];
const reversed = reverse(array);

console.log(reversed.next()); // { value: 5, done: false }
console.log(reversed.next()); // { value: 4, done: false }
console.log(reversed.next()); // { value: 3, done: false }
console.log(reversed.next()); // { value: 2, done: false }
console.log(reversed.next()); // { value: 1, done: false }
console.log(reversed.next()); // { value: undefined, done: true }
```
