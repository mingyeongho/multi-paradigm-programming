## 이터레이터

이터레이터는 **지연 평가**를 가능하게 해주는 도구.

**지연 평가**: 즉시 값을 변경시키는게 아니라 필요할 때 변경하는 방식.

e.g.

```ts
// 즉시 평가 - 호출 즉시 전체 배열에 계산이 수행됨.
const pow = [1, 2, 3].map((num) => num ** 2);

// 지연 평가
const pow = (array: Array<number>): Iterator<number> => {
  let idx = 0;
  return {
    next: () => {
      if (idx < array.length) {
        return {
          done: false,
          value: array[idx++] ** 2,
        };
      } else {
        return {
          done: true,
          value: undefined,
        };
      }
    },
  };
};

const nums = [1, 2, 3];
const powIter = pow(nums);

console.log(powIter.next()); // { done: false, value: 1 }
console.log(powIter.next()); // { done: false, value: 4 }
console.log(powIter.next()); // { done: false, value: 9 }
console.log(powIter.next()); // { done: true, value: undefined }
```

### 일급 함수

**값처럼** 사용할 수 있는 함수.

e.g.

- 함수를 매개변수로 사용
- 함수를 리턴값으로 사용
- 변수에 함수를 담을 수 있다.

### 고차 함수

함수를 매개변수로 받거나 리턴값으로 사용하는 함수

e.g. `map`, `filter`, `reduce` 등
